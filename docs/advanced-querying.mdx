## Building a Filtered Search

We can us this functionality to directly control the query with either relying on the "All Fliter" GUI . For this example, let's say we want to get iPhone products, but don't want to have any products that are categorized as accessories or cases.&#x20;

This can be done using the `Raw Categories` and searching for `phone`. As well we can filter out accessories by focusing on a minimum price. Setting `Most Recent Price Amount `to` 1000.`
