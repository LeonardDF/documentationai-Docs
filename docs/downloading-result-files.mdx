---
title: Downloading Result Files
---
With the Datafiniti Portal, downloading your results is a breeze!

<Steps>
  <Step title="Run the Download">
    <Callout kind="alert" collapsed="false">
      **Download File Format**

      Be sure to double check which file format you are downloading. **Once you trigger the download, you cannot change the format** and any records downloaded will count towards your total credits used.
    </Callout>

    Once you have run a query and generated your results, you can start a download by clicking on "Download."

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452373954-7mhfirclhdf-a822d35-DFPdownloadButton.png?fm=auto&auto=compress%2Cformat" width="2230" height="1290" alt="Download button in the Datafiniti Portal" />

    The "Download" button will prompt you with the "Download Records" dialog where you can:

    - Determine if you want JSON or CSV format
    - Set the number of records to download
    - Determine which specific view you'd like to use

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452375541-3yec1k9z3ck-f3adabe-DFPdownloadOptions.png?fm=auto&auto=compress%2Cformat" width="2230" height="1204" alt="Download options dialog showing format, record count, and view selection" />

    After you submit your download, you'll see the "Green box" prompt telling you that you will be redirected shortly to the downloaded results page.

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452378718-vnbnqnyypr-2a814e9-DFPdownloading.png?fm=auto&auto=compress%2Cformat" width="2230" height="1204" alt="Download in progress notification" />
  </Step>
  <Step title="Access Your Downloads">
    On the "Downloads" tab of your account, you can see all downloads you have requested. Clicking on the block of the download ID will take you to your specific downloaded records.

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452380509-quf7wf41cfq-cff6e0d-DFPdownloadPage.png?fm=auto&auto=compress%2Cformat" width="2230" height="1204" alt="Downloads page showing all requested downloads" />

    Clicking on "Download All" will provide you with more specifics regarding the query in the "query" field on this page. Information here is pertaining to this specific query. "Download All" will show the amount of records used in the "credits used" field.

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452382493-2l4vedb258t-56bcf7a-DFPDownloadedResults.png?fm=auto&auto=compress%2Cformat" width="2256" height="1204" alt="Downloaded results page with query details and credit usage" />

    It's as simple as that! You are now ready to use your data however you need!
  </Step>
</Steps>
