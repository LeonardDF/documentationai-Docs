---
title: Downloading Result Files
---
With the Datafiniti Portal, downloading your results is a breeze!

<Steps>
  <Step title="Run the Download" title-type="p">
    <Callout kind="alert" collapsed="false">
      **Download File Format**

      Be sure to double check which file format you are downloading. **Once you trigger the download, you cannot change the format** and any records downloaded will count towards your total credits used.
    </Callout>

    Once you have run a query and generated your results, you can start a download by clicking on "Download."

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1776887624287-pc1x1d9uq8l-Datafiniti-Portal-Property-click-download.gif?fm=auto&auto=compress%2Cformat" width="1698" height="1238" alt="" />



    The "Download" button will prompt you with the "Download Records" dialog, where you can:

    - Determine if you want JSON or CSV format

    - Set the number of records to download

    - Determine which specific view you'd like to use

    After you submit your download, you'll see the "Green box" prompt telling you that you will be redirected shortly to the downloaded results page.
  </Step>

  <Step title="Access Your Downloads" title-type="p">
    On the "Downloads" tab of your account, you can see all downloads you have requested. Clicking on the block of the download ID will take you to your specific downloaded records.



    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1776887724210-drgmu0156a7-Datafiniti-Portal-Settings-04-22-2026_02_54_PM.png?fm=auto&auto=compress%2Cformat" width="1718" height="1270" alt="" />

    Clicking on "Download All" will provide you with more specifics regarding the query in the "query" field on this page. Information here pertains to this specific query. "Download All" will show the number of records used in the "credits used" field.

    <Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1776887831064-l2mfjpz3x3r-Datafiniti-Portal-download-id-04-22-2026_02_56_PM.png?fm=auto&auto=compress%2Cformat" width="614" height="669" alt="" />

    It's as simple as that! You are now ready to use your data however you need!
  </Step>
</Steps>
