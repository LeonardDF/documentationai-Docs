---
title: Product Data Schema
---

The product data schema below lists all available fields in a product record.  Note that actual product records (actually, most of them) may not have every field present.

<ParamField body="asins" param-type="keyword" showLocation="false">
  The ASIN (Amazon identifier) used for this product.


  **Example:**
  ```json
  "asins":  "B0009XCFRE"
  ```
</ParamField>

<ParamField body="brand" param-type="text" showLocation="false">
  The brand name of this product as presented by the data source.


  **Example:**
  ```json
  "brand": "Sumsung Electronics"
  ```
</ParamField>

<ParamField body="categories" param-type="text" showLocation="false">
  A list of category keywords used for this product across multiple sources.

  **Example:**
  ```json
  "categories": [
     "Home Improvement",
     "Heating, Cooling, & Air Quality",
     "Heaters",
     "Propane Heaters"
   ]
  ```
</ParamField>

<ParamField body="canonicalBrand" param-type="text" showLocation="false">
  The canonical brand name after normalization of the name from the source.


  **Example:**
  ```json
  "canonicalBrand": "Sumsung"
  ```
</ParamField>

<ParamField body="colors" param-type="text" showLocation="false">
  A list of colors available for this product.

  **Example:**
  ```json
  "colors": [
     "White",
     "Black"
   ]
  ```
</ParamField>

<ParamField body="count" param-type="keyword" showLocation="false">
  The number of units included in the product packaging.  Can include a description of the unit.


  **Example:**
  ```json
  "count": "30"
  ```
</ParamField>

<ParamField body="dateAdded" param-type="date" showLocation="false">
  The date this product was first added to the product database.


  **Example:**
  ```json
  "dateAdded": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="dateUpdated" param-type="date" showLocation="false">
  The most recent date this product was updated or seen by our system.


  **Example:**
  ```json
  "dateUpdated": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="descriptions" param-type="nested" showLocation="false">
  A list of descriptions for this product from various sources.  Each description contains:

  **Example:**
  ```json
  "descriptions": [
     {
       "dateSeen": "2017-01-08T19:12:13Z",
       "sourceURLs": [    "https://www.domain.com/p/123"
       ],
       "value": "Now you can enjoy cooking
   your favorite food."
     },
     {
       "sourceURLs": [
    "http://www.website.com/4567"
       ],
       "value": "The Worthington Pro Grade 20 lb.
   Empty Propane Tank goes with your gas barbecue grill
   (sold separately) and accommodates 4.7 gallons of gas."
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="descriptions.dateSeen" param-type="date" showLocation="false">
      The date when this description has been seen selling this product.
    </ParamField>
    <ParamField body="descriptions.sourceURLs" param-type="keyword" showLocation="false">
      A list of source URLs where this specific description was seen.
    </ParamField>
    <ParamField body="descriptions.value" param-type="text" showLocation="false">
      A unique description for this product.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dimension" param-type="keyword" showLocation="false">
  The length, width, and height of this product.  Units included.
   [Dimensions Normalization](/docs/dimensions-normalization)

  **Example:**
  ```json
  "dimension": "23 in x 43.7 in x 30 in",`
   `"dimension": "16 ft x 22 ft",`
   `"dimension": "15 mm x 42 mm x 34 mm"
  ```
</ParamField>

<ParamField body="domains" param-type="domain" showLocation="false">
  A list of each unique domain found in the `sourceURLs` field.

  **Example:**
  ```json
  "domains": [
     "www.domain.com",
     "www.website.com"
   ]
  ```
</ParamField>

<ParamField body="ean" param-type="keyword" showLocation="false">
  The EAN codes for this product.  There may be more than one EAN in the list when a product has multiple variations,
    each using a different EAN.
     [Learn more about EANs](https://en.wikipedia.org/wiki/International_Article_Number).

  **Example:**
  ```json
  "ean": [
     "0014045125963",
     "0014045125965"
   ]
  ```
</ParamField>

<ParamField body="ean13" param-type="keyword" showLocation="false">
  The EAN-13 code for this product.    [Learn more about EANs](https://en.wikipedia.org/wiki/International_Article_Number).


  **Example:**
  ```json
  "ean13":"0014045125963"
  ```
</ParamField>

<ParamField body="ean8" param-type="keyword" showLocation="false">
  The EAN-8 code for this product.    [Learn more about EANs](https://en.wikipedia.org/wiki/International_Article_Number).


  **Example:**
  ```json
  "ean8":"96385074"
  ```
</ParamField>

<ParamField body="features" param-type="nested" showLocation="false">
  A list of features associated with this product.  The feature list can be collected from specific feature sections available on product listings for the product.  It can also serve as a catch-all field for miscellaneous characteristics of this product that don't make sense to include as top-level fields.
  
   Each feature contains:

  **Example:**
  ```json
  "features": [
     {
       "key": "Count",
       "replace": "true",
       "value": [
         "1"
       ]
     },
     {
       "key": "Included",
       "value": [
         "Batteries"
       ]
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="features.key" param-type="text" showLocation="false">
      A label for this feature.
    </ParamField>
    <ParamField body="features.replace" param-type="boolean" showLocation="false">
      Flag indicating value field is replaced versus appended.
    </ParamField>
    <ParamField body="features.value" param-type="text" showLocation="false">
      A list of possible values for this feature.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="financingAndLeasing" param-type="text" showLocation="false">
  A list of financing or leasing terms associated with this product.  Each term may contain:

  **Example:**
  ```json
  "financingAndLeasing": [
     {
       "additionalDetails": [
         "acquisitionFee: $925",
         "mileOverageFee: $0.30/mile"
       ],
       "apr": 4.15,
       "currency": "USD",
       "dateSeen": [
         "2018-09-20T04:00:00Z"
       ],
       "description": "Well equipped with features",
       "disclaimers": "Offer not valid in Puerto Rico.
   Lease financing available",
       "dueAtSigningMax": 7694,
       "dueAtSigningMin": 7694,
       "milesPerYear": 22500,
       "monthlyPaymentMax": 1269,
       "monthlyPaymentMin": 1269,
       "name": "2018 Civic Si Coupe 6
   Speed Manual Featured Special Lease",
       "offerStartDate": "2018-10-31T00:00:00.000Z",
       "offerEndDate": "2018-09-05T00:00:00.000Z",
       "term": "36 months",
       "type": "leasing",
       "securityDepositMax": 0,
       "securityDepositMin": 0,
       "sourceURLs": [
       ]
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="financingAndLeasing.additionalDetails" param-type="text" showLocation="false">
      Any additional information related to this term.
    </ParamField>
    <ParamField body="financingAndLeasing.apr" param-type="text" showLocation="false">
      The annual percentage rate.
    </ParamField>
    <ParamField body="financingAndLeasing.currency" param-type="keyword" showLocation="false">
      The currency listed for any monetary values.
    </ParamField>
    <ParamField body="financingAndLeasing.dateSeen" param-type="date" showLocation="false">
      A list of dates when this term was seen.
    </ParamField>
    <ParamField body="financingAndLeasing.description" param-type="text" showLocation="false">
      The description provided for this term.
    </ParamField>
    <ParamField body="financingAndLeasing.disclaimers" param-type="text" showLocation="false">
      Any disclaimers associated with this term.
    </ParamField>
    <ParamField body="financingAndLeasing.dueAtSigningMax" param-type="float" showLocation="false">
      The maximum monetary amount due when purchasing this product under this term.
    </ParamField>
    <ParamField body="financingAndLeasing.dueAtSigningMin" param-type="float" showLocation="false">
      The minimum monetary amount due when purchasing this product under this term.
    </ParamField>
    <ParamField body="financingAndLeasing.milesPerYear" param-type="float" showLocation="false">
      Usually associated with vehicles, the maximum mileage allowed under this term.
    </ParamField>
    <ParamField body="financingAndLeasing.monthlyPaymentMax" param-type="float" showLocation="false">
      The maximum amount due each month under this term.
    </ParamField>
    <ParamField body="financingAndLeasing.monthlyPaymentMin" param-type="float" showLocation="false">
      The minimum amount due each month under this term.
    </ParamField>
    <ParamField body="financingAndLeasing.name" param-type="text" showLocation="false">
      The name for this term.
    </ParamField>
    <ParamField body="financingAndLeasing.type" param-type="keyword" showLocation="false">
      Either `financing` or `leasing`.
    </ParamField>
    <ParamField body="financingAndLeasing.securityDepositMax" param-type="float" showLocation="false">
      The maximum security deposit for this term.
    </ParamField>
    <ParamField body="financingAndLeasing.securityDepositMin" param-type="float" showLocation="false">
      The minimum security deposit for this term.
    </ParamField>
    <ParamField body="financingAndLeasing.sourceURLs" param-type="keyword" showLocation="false">
      A list of URLs where this term was seen.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="flavors" param-type="text" showLocation="false">
  A list of flavors available for this product.

  **Example:**
  ```json
  "flavors": [
     "Fruit Punch",
     "Berry",
     "Grape"
   ]
  ```
</ParamField>

<ParamField body="gtins" param-type="keyword" showLocation="false">
  A list of all Global Trade Identifier Numbers for this product. Values will include all ean, isbn, and upc values.
    [Learn more about GTINs](https://en.wikipedia.org/wiki/Global_Trade_Item_Number).

  **Example:**
  ```json
  "gtins": [
     "0014045125963",
     "0014045125965",
     "014045125963",
     "014045125961",
     "425261",
     "3882215542"
   ]
  ```
</ParamField>

<ParamField body="imageURLs" param-type="keyword" showLocation="false">
  A list of image URLs for this product.

  **Example:**
  ```json
  "imageURLs": [
     "https://i5.images.com/asr.jpeg"
   ]
  ```
</ParamField>

<ParamField body="isbn" param-type="keyword" showLocation="false">
  The ISBN code for this product.  [Learn more about ISBNs](https://en.wikipedia.org/wiki/International_Standard_Book_Number).


  **Example:**
  ```json
  "isbn": "3882215542"
  ```
</ParamField>

<ParamField body="keys" param-type="keyword" showLocation="false">
  A list of internal Datafiniti identifiers for this product.  The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.  [Learn how this works](/docs/how-product-records-are-merged)

  **Example:**
  ```json
  "keys": [
     "014045125963",
     "0014045125963",
     "worthington/299581"
   ]
  ```
</ParamField>

<ParamField body="manufacturer" param-type="text" showLocation="false">
  The manufacturer of this product.


  **Example:**
  ```json
  "manufacturer": "Worthington"
  ```
</ParamField>

<ParamField body="manufacturerNumber" param-type="keyword" showLocation="false">
  The manufacturer or model number of this product.


  **Example:**
  ```json
  "manufacturerNumber": "299581"
  ```
</ParamField>

       "province": "TX"

       "dateSeen": [
         "2017-01-08T19:12:13Z"
       ],
       "name": "Next Day Mro, Llc"
     }
    ]

      The merchant's city.

      True if the merchant is an individual. False if it's a company.

    <ParamField body="merchants.name" param-type="text" showLocation="false">
      The merchant's name.
    </ParamField>
    <ParamField body="merchants.phone" param-type="keyword" showLocation="false">
      The merchant's phone number.

    <ParamField body="merchants.postalCode" param-type="text" showLocation="false">
