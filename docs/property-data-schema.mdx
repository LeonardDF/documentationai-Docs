---
title: Property Data Schema
---

The property data schema below lists all available fields in a property record. Note that actual property records (actually, most of them) may not have every field present.

<ParamField body="absenteeOwner" param-type="nested" showLocation="false">
  The owner of a property who does not reside in or otherwise occupy that property.

  **Example:**
  ```json
  "absenteeOwner": [
      {
        "absent": false,
        "firstDateSeen": "2024-04-11T17:17:53.270Z",
        "lastDateSeen": "2024-04-11T17:17:53.270Z"
      }
    ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="absenteeOwner.absent" param-type="boolean" showLocation="false">
      Is currently an absentee owner
    </ParamField>
    <ParamField body="absenteeOwner.firstDateSeen" param-type="date" showLocation="false">
      Earliest date seen of the absentee owner.
    </ParamField>
    <ParamField body="absenteeOwner.lastDateSeen" param-type="date" showLocation="false">
      Latest date seen of the absentee owner.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="address" param-type="text" showLocation="false">
  The physical street address for this property.


  **Example:**
  ```json
  "address": "7206 Walling Ln"
  ```
</ParamField>

<ParamField body="appliances" param-type="text" showLocation="false">
  The appliances included with the property.


  **Example:**
  ```json
  "appliances": ["Cooktop","Dryer","Dishwasher","Microwave","Refrigerator","Washer"]
  ```
</ParamField>

<ParamField body="architecturalStyles" param-type="text" showLocation="false">
  The architectural style of the property.


  **Example:**
  ```json
  "architecturalStyles": ["Ranch","Bungalow"]
  ```
</ParamField>

<ParamField body="assessedValues" param-type="nested" showLocation="false">
  The assessed value of the property. Each assessed value may contain:

  **Example:**
  ```json
  "assessedValues": [
      {
        "totalAmount": 1745400.0,
        "improvementsAmount": 1745400.0,
        "landAmount": 1745400.0,
        "year": 1985
      }
    ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="assessedValues.improvementsAmount" param-type="float" showLocation="false">
      assessed value for improvements made to the property.
    </ParamField>
    <ParamField body="assessedValues.landAmount" param-type="float" showLocation="false">
      assessed value for the land/lot of the property.
    </ParamField>
    <ParamField body="assessedValues.totalAmount" param-type="float" showLocation="false">
      amount value the property is currently assessed for.
    </ParamField>
    <ParamField body="assessedValues.year" param-type="integer" showLocation="false">
      year value when the property was assessed.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="brandTenant" param-type="string" showLocation="false">

  **Example:**
  ```json
  "brandTenant": "Bob Evans Restaurants, LLC"
  ```
</ParamField>

<ParamField body="brokers" param-type="nested" showLocation="false">
  A list of brokers that have represented this property over time. Each broker may contain:

  **Example:**
  ```json
  "brokers": {
      "emails": [
        "tomatterson@realtracs.com"
      ],
      "licenseNumber": "1059280",
      "websites": [
        "http://www.fridrichandclark.com/"
      ],
      "phones": [
        "6153511737",
        "6153274800"
      ],
      "agent": "Tom Atterson",
      "firstName": "Tom",
      "lastName": "Atterson",
      "sourceURLs": [
        "https://datafiniti.co/bd-datafiniti",
        "https://www.redfin.com/TN/Arrington/4301-Union-Springs-Ln-37014/home/109775697"
      ],
      "dateSeen": "2025-03-28T17:18:49.715Z",
      "company": "Frich & Clare Realty",
      "people_key": "tom/atterson/1364043647"
    }
  ```

  <Expandable title="Sub-fields">
    <ParamField body="brokers.agent" param-type="text" showLocation="false">
      The name of the real estate agent representing this property.
    </ParamField>
    <ParamField body="brokers.company" param-type="text" showLocation="false">
      The company this agent works for (or the company representing this property).
    </ParamField>
    <ParamField body="brokers.dateSeen" param-type="date" showLocation="false">
      The date this agent/company was seen representing the property.
    </ParamField>
    <ParamField body="brokers.emails" param-type="keyword" showLocation="false">
      A list of email addresses associated with the agent or company.
    </ParamField>
    <ParamField body="brokers.licenseNumber" param-type="text" showLocation="false">
      The license number of the agent representing the property.
    </ParamField>
    <ParamField body="brokers.licenseType" param-type="text" showLocation="false">
      The license Type of the agent representing the property.
    </ParamField>
    <ParamField body="brokers.people_key" param-type="text" showLocation="false">
      The corresponding key that link to Datafiniti's people data record. (contact, demographics, & more)
    </ParamField>
    <ParamField body="brokers.phones" param-type="keyword" showLocation="false">
      A phone number associated with the agent or company.
    </ParamField>
    <ParamField body="brokers.websites" param-type="keyword" showLocation="false">
      A list of websites associated with the agent or company.
    </ParamField>
    <ParamField body="brokers.workEmail" param-type="keyword" showLocation="false">
      Email address specifically associated with the business the broker works for currently.

  </Expandable>
</ParamField>
