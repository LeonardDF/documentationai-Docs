## How to Search for a Restaurant by Cuisine

One of the more common use cases we see is customers searching restaurants by cuisine type. For this example, we will search Austin Texas for Tex-Mex restaurants.

## 2. Searching by Cuisine via API Tab

You can copy and paste the query from the API Call and use it in the API tab

<Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1775452638165-x56lwxmfr0a-fccf686-Datafiniti-Portal-Business-texmex-API.png?fm=auto&auto=compress%2Cformat" width="1920" height="1080" alt="Screenshot showing 2. searching by cuisine via api tab" noCaption />

Simply click the API tab and paste your query into the "start typing" box.
