## 2. Adding review filters to the previous search

You can refine the cuisine search from the previous section by using the portal filters to add review criteria.

For example, if you want Tex-Mex restaurants with ratings between 4 and 5, add a review rating filter in the visual query builder and keep the same location and cuisine criteria:

<Image src="https://blob-cdn.documentation.ai/org-a829270c-4408-45aa-90cb-a8cad2967955/doc-82f0e261-4f6b-45af-9f00-b0b805bf07d2/1777310076218-4ro46wt97t6-pasted-image-business-reviewrating.png?fm=auto&auto=compress%2Cformat" width="859" height="1050" alt="" />
