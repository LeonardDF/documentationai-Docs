---
title: Business Data Schema
---

The business data schema below lists all available fields in a business record.  Note that actual business records (actually, most of them) may not have every field present.

<ParamField body="address" param-type="text" showLocation="false">
  The physical street address for this business location.


  **Example:**
  ```json
  "address": "3382 30th St"
  ```
</ParamField>

<ParamField body="careerPageURL" param-type="text" showLocation="false">
  A URL pointing to the career page for the business.


  **Example:**
  ```json
  "careerPageURL": "http://www.searocketbistro.com/employment"
  ```
</ParamField>

<ParamField body="categories" param-type="text" showLocation="false">
  A list of category keywords used for this business across multiple sources.

  **Example:**
  ```json
  "categories": [
     "Seafood Restaurant",
     "Fish & Seafood Retail",
     "restaurant"
   ]
  ```
</ParamField>

<ParamField body="city" param-type="text" showLocation="false">
  The city of this business location.


  **Example:**
  ```json
  "city": "San Diego"
  ```
</ParamField>

<ParamField body="claimed" param-type="text" showLocation="false">
  A list of websites where the business location has been claimed by the business' owner.

  **Example:**
  ```json
  "claimed": [
     "http://yellowpages.com/",
     "http://foursquare.com/"
   ]
  ```
</ParamField>

<ParamField body="country" param-type="text" showLocation="false">
  The two-letter country code for the business location's country.


  **Example:**
  ```json
  "country": "US"
  ```
</ParamField>

<ParamField body="cuisines" param-type="text" showLocation="false">
  A list of cuisines offered by this business location.  This field will only show for restaurants or other eating locations.

  **Example:**
  ```json
  "cuisines": [
     "Seafood Restaurants",
     "Sandwich Shop",
     "Seafood, Vegetarian / Vegan, Mediterranean"
   ]
  ```
</ParamField>

<ParamField body="dateAdded" param-type="text" showLocation="false">
  The date this business was first added to the business database.


  **Example:**
  ```json
  "dateAdded": "2016-03-24T09:27:02Z"
  ```
</ParamField>

<ParamField body="dateUpdated" param-type="text" showLocation="false">
  The most recent date this business was updated or seen by our system.


  **Example:**
  ```json
  "dateUpdated": "2017-03-20T20:45:49Z"
  ```
</ParamField>

<ParamField body="descriptions" param-type="nested" showLocation="false">
  A list of descriptions for this business from various sources.  Each description contains:

  **Example:**
  ```json
  "descriptions": [
     {
       "dateSeen": [
         "2016-06-18T01:17:41Z"
       ],
       "sourceURLs": [
         "https://www.zagat.com/r/petes-seafood-and-sandwich-san-diego"
       ],
       "value": "Sizable made-to-order sandwiches, including lobster rolls, meatball Parmesan heroes and a variety of vegetarian options, are served at this East Coast–style sandwich shop nestled in the heart of North Park."
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="descriptions.dateSeen" param-type="text" showLocation="false">
      A list of dates when this description has been seen for this business.
    </ParamField>
    <ParamField body="descriptions.sourceURLs" param-type="text" showLocation="false">
      A list of source URLs where this specific description was seen.
    </ParamField>
    <ParamField body="descriptions.value" param-type="text" showLocation="false">
      A unique description for this business.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="emails" param-type="text" showLocation="false">
  A list of emails belonging to the business.

  **Example:**
  ```json
  "emails": [
     "elena@searocketbistro.com"
   ]
  ```
</ParamField>

<ParamField body="facebookPageURL" param-type="text" showLocation="false">
  A URL pointing to the Facebook page for the business.


  **Example:**
  ```json
  "facebookPageURL": "http://www.facebook.com/pages/Sea-Rocket-Bistro/114213985293774"
  ```
</ParamField>

<ParamField body="faxes" param-type="text" showLocation="false">
  A list of fax numbers for the business.

  **Example:**
  ```json
  "faxes": [
     "3135619486"
   ]
  ```
</ParamField>

<ParamField body="features" param-type="nested" showLocation="false">
  A list of features associated with this business location.  The feature list can be collected from specific feature sections available on listings for this business.  It can also serve as a catch-all field for miscellaneous characteristics of this business that don't make sense to include as top-level fields.
  
   Each feature contains:

  **Example:**
  ```json
  "features": [
     {
       "key": "attire",
       "value": [
         "Casual Dress"
       ]
     },
     {
       "key": "parking",
       "value": [
         "Street Parking",
         "Valet"
       ]
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="features.key" param-type="text" showLocation="false">
      A label for this feature.
    </ParamField>
    <ParamField body="features.value" param-type="text" showLocation="false">
      A list of possible values for this feature.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="hours" param-type="nested" showLocation="false">
  The hours of operation for this business location.
  
   Each hour object may contain:

  **Example:**
  ```json
  "hours": [
     {
       "day": "Mon - Fri",
       "hour": "11:00 am - 9:00 pm"
     },
     {
       "day": "Sat - Sun",
       "hour": "11:00 am - 10:00 pm"
     },
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="hours.day" param-type="text" showLocation="false">
      The day(s) for this hour of operation.
    </ParamField>
    <ParamField body="hours.dept" param-type="text" showLocation="false">
      The department or business unit related to this hour of operation.
    </ParamField>
    <ParamField body="hours.hour" param-type="text" showLocation="false">
      The hours of operation.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="imageURLs" param-type="text" showLocation="false">
  A list of image URLs associated with this business.

  **Example:**
  ```json
  "imageURLs": [
     "https://irs0.4sqi.net/img/general/200x200/118688231_EmfdW9_8FpBrxsa5WDVektvkubj-XpCW5ln6s6pnClo.jpg",
     "https://irs0.4sqi.net/img/general/200x200/1556291_gjOF8nnudHh_ICSWXoTs75_FxtcKt25nZtwbqjaHtKQ.jpg",
   ]
  ```
</ParamField>

<ParamField body="isClosed" param-type="text" showLocation="false">
  A `true` or `false` value for whether or not the business location is closed.


  **Example:**
  ```json
  "isClosed": "false"
  ```
</ParamField>

<ParamField body="keys" param-type="text" showLocation="false">
  A list of internal Datafiniti identifiers for this business.  The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.  [Learn more about how this works](https://datafiniti.co).

  **Example:**
  ```json
  "keys": [
     "us/ca/sandiego/338230thst/2075732923"
   ]
  ```
</ParamField>

<ParamField body="languagesSpoken" param-type="text" showLocation="false">
  A list of the languages spoken at this business.

  **Example:**
  ```json
  "languagesSpoken: [
     "English",
     "Spanish"
   ]
  ```
</ParamField>

<ParamField body="latitude" param-type="text" showLocation="false">

<ParamField body="longitude" param-type="text" showLocation="false">

</ParamField>

<ParamField body="menuPageURL" param-type="text" showLocation="false">

<ParamField body="menus" param-type="nested" showLocation="false">
  A list of menu items offered by the business.  This field will only show for restaurants or other eating locations.
  
   Each menu item may include:

  **Example:**
  ```json
  "menus": [
     {
       "amountMin": 12,
       "amountMax": 12,
       "category": "sample menu",
       "currency": "USD",
       "dateSeen": [
         "2017-03-22T01:03:56Z"
       ],
       "name": "Fish Carpaccio",
       "sourceURLs": [
     "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm"
       ]
     },
     {
       "amountMin": 7,
       "amountMax": 7,
       "category": "sample menu",
       "currency": "USD",
       "dateSeen": [
         "2017-03-22T01:03:56Z"
       ],
       "description": "A dense bread pudding recipe served cold with butterscotch sauce",
       "name": "Seasonal Pumpkin Bread Pudding",
       "sourceURLs": [
     "http://www.superpages.com/bp/san-diego-ca/petes-seafood-sandwich-L2615943735.htm"

     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="menus.amountMin" param-type="text" showLocation="false">
      The minimum price value of the item.
    </ParamField>
    <ParamField body="menus.amountMax" param-type="text" showLocation="false">
      The maximum price value of the item.  This is typically the same as amountMin, except in cases where a price range is listed.
    </ParamField>
    <ParamField body="menus.category" param-type="text" showLocation="false">
      The menu section this item belongs to.
    </ParamField>
    <ParamField body="menus.currency" param-type="text" showLocation="false">
      The currency listed for amountMin and amountMax.
    </ParamField>
    <ParamField body="menus.dateSeen" param-type="text" showLocation="false">
      A list of dates when this item was seen.
    </ParamField>
    <ParamField body="menus.description" param-type="text" showLocation="false">
      A description for the item.
    </ParamField>
    <ParamField body="menus.name" param-type="text" showLocation="false">
      The item's name on the menu.
    </ParamField>
    <ParamField body="menus.sourceURLs" param-type="text" showLocation="false">
      A list of URLs where this item was seen.

  </Expandable>
