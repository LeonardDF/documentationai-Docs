---
title: Product Data Schema
---

The product data schema below lists all available fields in a product record.  Note that actual product records (actually, most of them) may not have every field present.

<ParamField body="asins" param-type="text" showLocation="false">
  A list of ASINs (Amazon identifiers) used for this product.

  **Example:**
  ```json
  "asins": [
     "B0009XCFRE"
   ]
  ```
</ParamField>

<ParamField body="brand" param-type="text" showLocation="false">
  The brand name of this product.


  **Example:**
  ```json
  "brand": "Worthington"
  ```
</ParamField>

<ParamField body="categories" param-type="text" showLocation="false">
  A list of category keywords used for this product across multiple sources.

  **Example:**
  ```json
  "categories": [
     "Home Improvement",
     "Heating, Cooling, & Air Quality",
     "Heaters",
     "Propane Heaters"
   ]
  ```
</ParamField>

<ParamField body="colors" param-type="text" showLocation="false">
  A list of colors available for this product.

  **Example:**
  ```json
  "colors": [
     "White",
     "Black"
   ]
  ```
</ParamField>

<ParamField body="count" param-type="text" showLocation="false">
  The number of units included in the product packaging.


  **Example:**
  ```json
  "count": 1
  ```
</ParamField>

<ParamField body="dateAdded" param-type="text" showLocation="false">
  The date this product was first added to the product database.


  **Example:**
  ```json
  "dateAdded": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="dateUpdated" param-type="text" showLocation="false">
  The most recent date this product was updated or seen by our system.


  **Example:**
  ```json
  "dateUpdated": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="descriptions" param-type="nested" showLocation="false">
  A list of descriptions for this product from various sources.  Each description contains:

  **Example:**
  ```json
  "descriptions": [
     {
       "dateSeen": [
         "2017-01-08T19:12:13Z"
       ]
       "sourceURLs": [
         "https://www.domain.com/p/123"
       ],
       "value": "Now you can enjoy cooking your favorite food."
     },
     {
       "sourceURLs": [
         "http://www.website.com/4567"
       ],
       "value": "The Worthington Pro Grade 20 lb. Empty Propane Tank goes with your gas barbecue grill (sold separately) and accommodates 4.7 gallons of gas."
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="descriptions.dateSeen" param-type="text" showLocation="false">
      A list of dates when this description has been seen selling this product.
    </ParamField>
    <ParamField body="descriptions.sourceURLs" param-type="text" showLocation="false">
      A list of source URLs where this specific description was seen.
    </ParamField>
    <ParamField body="descriptions.value" param-type="text" showLocation="false">
      A unique description for this product.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="dimension" param-type="text" showLocation="false">
  The length, width, and height of this product.  Units included.


  **Example:**
  ```json
  "dimension": "23 in x 43.7 in x 30 in"
  ```
</ParamField>

<ParamField body="ean" param-type="text" showLocation="false">
  The EAN codes for this product.  There may be more than one EAN in the list when a product has multiple variations,
    each using a different EAN.
     [Learn more about EANs](https://en.wikipedia.org/wiki/International_Article_Number).

  **Example:**
  ```json
  "ean": [
     "0014045125963",
     "0014045125965"
   ]
  ```
</ParamField>

<ParamField body="features" param-type="nested" showLocation="false">
  A list of features associated with this product.  The feature list can be collected from specific feature sections available on product listings for the product.  It can also serve as a catch-all field for miscellaneous characteristics of this product that don't make sense to include as top-level fields.
  
   Each feature contains:

  **Example:**
  ```json
  "features": [
     {
       "key": "Count",
       "value": [
         "1"
       ]
     },
     {
       "key": "Dimensions",
       "value": [
         "12.5 x 12.5 x 18.5 in"
       ]
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="features.key" param-type="text" showLocation="false">
      A label for this feature.
    </ParamField>
    <ParamField body="features.value" param-type="text" showLocation="false">
      A list of possible values for this feature.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="flavors" param-type="text" showLocation="false">
  A list of flavors available for this product.

  **Example:**
  ```json
  "flavors": [
     "Fruit Punch",
     "Berry",
     "Grape"
   ]
  ```
</ParamField>

<ParamField body="imageURLs" param-type="text" showLocation="false">
  A list of image URLs for this product.

  **Example:**
  ```json
  "imageURLs": [
     "https://i5.images.com/asr.jpeg"
   ]
  ```
</ParamField>

<ParamField body="isbn" param-type="text" showLocation="false">
  The ISBN code for this product.  [Learn more about ISBNs](https://en.wikipedia.org/wiki/International_Standard_Book_Number).


  **Example:**
  ```json
  "isbn": "3882215542"
  ```
</ParamField>

<ParamField body="keys" param-type="text" showLocation="false">
  A list of internal Datafiniti identifiers for this product.  The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.  [Learn more about how this works](https://datafiniti.co).

  **Example:**
  ```json
  "keys": [
     "014045125963",
     "0014045125963",
     "worthington/299581"
   ]
  ```
</ParamField>

<ParamField body="manufacturer" param-type="text" showLocation="false">
  The manufacturer of this product.


  **Example:**
  ```json
  "manufacturer": "Worthington"
  ```
</ParamField>

<ParamField body="manufacturerNumber" param-type="text" showLocation="false">
  The manufacturer or model number of this product.


  **Example:**
  ```json
  "manufacturerNumber": "299581"
  ```
</ParamField>

<ParamField body="merchants" param-type="nested" showLocation="false">
  A list of merchants selling this product.  These are typically third-party merchants found on e-commerce websites.
  
   Merchant objects can contain:

  **Example:**
  ```json
  "merchants": [
     {
       "address": "2801 S Ih 35",
       "availability": "...",
       "city": "Austin",
       "country": "US",
       "dateSeen": [
         "2017-01-08T19:12:13Z"
       ],
       "isPrivateSeller": "false"
       "name": "Zappysales",
       "phone": "5127131234",
       "postalCode": "78701",
       "province": "TX"
     },
     {
       "dateSeen": [
         "2017-01-08T19:12:13Z"
       ],
       "name": "Next Day Mro, Llc"
     }
    ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="merchants.address" param-type="text" showLocation="false">
      The merchant's physical address.
    </ParamField>
    <ParamField body="merchants.availability" param-type="text" showLocation="false">
      A description of the product's availability with this merchant.
    </ParamField>
    <ParamField body="merchants.city" param-type="text" showLocation="false">
      The merchant's city.
    </ParamField>
    <ParamField body="merchants.dateSeen" param-type="text" showLocation="false">
      A list of dates when this merchant has been seen selling this product.
    </ParamField>
    <ParamField body="merchants.isPrivateSeller" param-type="text" showLocation="false">
      True if the merchant is an individual. False if it's a company.
    </ParamField>
    <ParamField body="merchants.name" param-type="text" showLocation="false">
      The merchant's name.
    </ParamField>
    <ParamField body="merchants.phone" param-type="text" showLocation="false">
      The merchant's phone number.
    </ParamField>
    <ParamField body="merchants.postalCode" param-type="text" showLocation="false">
      The merchant's postal or zip code.
    </ParamField>
    <ParamField body="merchants.province" param-type="text" showLocation="false">
      The merchant's province or state.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="name" param-type="text" showLocation="false">
  The product's name.


  **Example:**
  ```json
  "name": "Worthington 20-lb Tank"
  ```
</ParamField>

<ParamField body="prices" param-type="nested" showLocation="false">
  A list of prices for this product.
  
   Price objects may contain:

  **Example:**
  ```json
  "prices": [
     {
       "amountMin": 59.96,
       "amountMax": 59.96,
       "availability": "...",
       "color": "...",
       "condition": "New",
       "count": "...",
       "currency": "USD",
       "dateAdded": "2017-01-08T19:12:13Z",
       "dateSeen": [
         "2016-06-10T00:00:00Z"
       ],
       "flavor": "...",
       "isSale": "false",
       "isSold": "false",
       "merchant": "domain.com",
       "offer": "22% off",
       "returnPolicy": "90-Day",
       "shipping": "FREE shipping",
       "size": "...",
       "sourceURLs": [
         "https://www.domain.com/p/123",
        "warranty": "..."
       ]
     },
     {
       "amountMin": 62.06,
       "amountMax": 72.06,
       "condition": "new",
       "currency": "USD",
       "dateAdded": "2017-01-08T19:12:13Z",
       "dateSeen": [
         "2016-10-28T00:00:00Z"
       ],
       "merchant": "MRO - website.com",
       "sourceURLs": [
         "http://www.website.com/4567"
       ]
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="prices.amountMin" param-type="text" showLocation="false">
      The minimum price value listed.
    </ParamField>
    <ParamField body="prices.amountMax" param-type="text" showLocation="false">
      The maximum price value listed.  This is typically the same as amountMin, except in cases where a price range is listed.
    </ParamField>
    <ParamField body="prices.availability" param-type="text" showLocation="false">
      The product's availability at this price.
    </ParamField>
    <ParamField body="prices.color" param-type="text" showLocation="false">
      The color associated with this price.
    </ParamField>
    <ParamField body="prices.condition" param-type="text" showLocation="false">
      The condition of the product when being sold at this price.
    </ParamField>
    <ParamField body="prices.currency" param-type="text" showLocation="false">
      The currency listed for amountMin and amountMax.
    </ParamField>
    <ParamField body="prices.dateAdded" param-type="text" showLocation="false">
      The date this price was first added to the prices list.
    </ParamField>
    <ParamField body="prices.dateSeen" param-type="text" showLocation="false">
      A list of dates when this price was seen.
    </ParamField>
    <ParamField body="prices.flavor" param-type="text" showLocation="false">
      The flavor associated with this price.
    </ParamField>
    <ParamField body="prices.isSale" param-type="text" showLocation="false">
      A true/false for whether or not this price is a sale/discounted price.
    </ParamField>
    <ParamField body="prices.isSold" param-type="text" showLocation="false">
      A true/flase for whether or not the product has been sold at this price (typically reserved for vehicle listings).
    </ParamField>
    <ParamField body="prices.merchant" param-type="text" showLocation="false">
      The merchant and/or website selling at this price.
    </ParamField>
    <ParamField body="prices.offer" param-type="text" showLocation="false">
      Any special offer associated with this price.
    </ParamField>
    <ParamField body="prices.returnPolicy" param-type="text" showLocation="false">
      The return policy associated with this price.
    </ParamField>
    <ParamField body="prices.shipping" param-type="text" showLocation="false">
      The shipping terms associated with this price.
    </ParamField>
    <ParamField body="prices.sourceURLs" param-type="text" showLocation="false">
      A list of URLs where this price was seen.
    </ParamField>
    <ParamField body="prices.warranty" param-type="text" showLocation="false">
      The warranty associated with this price.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="quantities" param-type="nested" showLocation="false">
  A list of available quantities for this product.  Quantities indicate how many units of the product are available from a specific source at a given time.
  
   Quantity objects may contain:

  **Example:**
  ```json
  "quantities": [
     {
       "dateSeen": [
         "2013-06-02T16:00:00Z",
         "2014-05-01T14:00:00Z"
       ],
       "sourceURLs": [
         "http://www.website.com/4567"
       ],
       "value": 20
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="quantities.dateSeen" param-type="text" showLocation="false">
      A list of dates when this quantity was seen.
    </ParamField>
    <ParamField body="quantities.sourceURLs" param-type="text" showLocation="false">
      A list of URLs where this quantity was seen.
    </ParamField>
    <ParamField body="quantities.value" param-type="text" showLocation="false">
      The actual quantity value.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sizes" param-type="text" showLocation="false">
  A list of sizes available for this product.

  **Example:**
  ```json
  "sizes": [
     "Large",
     "Medium",
     "Small"
   ]
  ```
</ParamField>

<ParamField body="skus" param-type="nested" showLocation="false">
  A list of SKUs for this product.  SKUs are typically specific to individual retailers or websites.
  
   SKU objects may contain:

  **Example:**
  ```json
  "skus": [
     {
       "sourceURLs": [
         "http://www.web.com/p/123"
       ],
       "value": "1234XK6"
     }
   ]
  ```

  <Expandable title="Sub-fields">
    <ParamField body="skus.sourceURLs" param-type="text" showLocation="false">
      A list of URLs where this SKU was seen.
    </ParamField>
    <ParamField body="skus.value" param-type="text" showLocation="false">
      The actual SKU.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="sourceURLs" param-type="text" showLocation="false">
  A list of URLs used to generate data for this product.

  **Example:**
  ```json
  "sourceURLs": [
     "https://www.domain.com/p/123",
     "http://www.website.com/4567"
   ]
  ```
</ParamField>

<ParamField body="upc" param-type="text" showLocation="false">
  The UPC code for this product.
     There may be more than one UPC in the list when a product has multiple variations,
    each using a different UPC.  [Learn more about UPCs](https://en.wikipedia.org/wiki/Universal_Product_Code).

  **Example:**
  ```json
  "upc": [
     "014045125963",
     "014045125961"
   ]
  ```
</ParamField>

<ParamField body="vin" param-type="text" showLocation="false">
  The VIN code for this product.  This is specific to car listings.  [Learn more about VINs](https://en.wikipedia.org/wiki/Vehicle_identification_number).


  **Example:**
  ```json
  "vin": "1FTMF1ET8EFB00361"
  ```
</ParamField>

<ParamField body="websiteIDs" param-type="text" showLocation="false">
  A list of website IDs for this product.  Website IDs are tied to a specific online retailer.  They are the unique identifier used by website.  They are used by Datafiniti when other unique identifiers are not available for a product.

  **Example:**
  ```json
  "websiteIDs": [
     "domain.com-123",
     "website.com-4567"
   ]
  ```
</ParamField>

<ParamField body="weight" param-type="text" showLocation="false">
  The weight of the product.  Units included.


  **Example:**
  ```json
  "weight": "20 lbs"
  ```
</ParamField>
