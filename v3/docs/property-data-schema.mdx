---
title: Property Data Schema
---

The property data schema below lists all available fields in a property record. Note that actual property records may not have every field present.

For a list of possible values for specific property fields, see the [Possible Values for Property Fields](/v3/docs/possible-values-for-property-fields) page.

<ParamField body="address" param-type="text" showLocation="false">
  The physical street address for this property.


  **Example:**
  ```json
  "address": "7206 Walling Ln"
  ```
</ParamField>

<ParamField body="availableDates" param-type="nested" showLocation="false">
  A list of dates this property is available (typically for rent). Each availability contains: `dateSeen`, `endDate`, `sourceURLs`, `startDate`.

  **Example:**
  ```json
  "availableDates": [{"dateSeen": ["2017-03-16T00:00:00Z"], "endDate": "2017-02-16T00:00:00Z", "sourceURLs": ["http://..."], "startDate": "2017-02-16T00:00:00Z"}]
  ```
</ParamField>

<ParamField body="brokers" param-type="nested" showLocation="false">
  A list of brokers that have represented this property over time. Each broker may contain: `agent`, `company`, `dateSeen`, `emails`, `phones`, `websites`.

  **Example:**
  ```json
  "brokers": [{"agent": "Frances Zuckerman", "company": "Fillmore", "dateSeen": ["2017-01-17T09:47:16Z"], "emails": ["fzuckerman@gmail.com"], "phones": ["3214561234"]}]
  ```
</ParamField>

<ParamField body="buildingName" param-type="text" showLocation="false">
  The name of the building where this property is located. This field is specific to apartments, condos, and similar property types.


  **Example:**
  ```json
  "buildingName": "The Grand"
  ```
</ParamField>

<ParamField body="city" param-type="text" showLocation="false">
  The city where the property is located.


  **Example:**
  ```json
  "city": "Dallas"
  ```
</ParamField>

<ParamField body="country" param-type="text" showLocation="false">
  The two-letter country code for the property location's country.


  **Example:**
  ```json
  "country": "US"
  ```
</ParamField>

<ParamField body="dateAdded" param-type="text" showLocation="false">
  The date this property was first added to the property database.


  **Example:**
  ```json
  "dateAdded": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="dateUpdated" param-type="text" showLocation="false">
  The most recent date this property was updated or seen by our system.


  **Example:**
  ```json
  "dateUpdated": "2017-01-08T19:12:13Z"
  ```
</ParamField>

<ParamField body="deposits" param-type="nested" showLocation="false">
  A list of deposit fees that have been seen for this property. Each deposit contains: `amount`, `currency`, `dateSeen`, `sourceURLs`.

  **Example:**
  ```json
  "deposits": [{"amount": 104, "currency": "USD", "dateSeen": ["2016-08-07T18:07:05Z"], "sourceURLs": ["https://..."]}]
  ```
</ParamField>

<ParamField body="descriptions" param-type="nested" showLocation="false">
  A list of descriptions for this property from various sources. Each description contains: `dateSeen`, `sourceURLs`, `value`.


  **Example:**
  ```json
  "descriptions": [{"sourceURLs": ["http://..."], "value": "charming starter home priced to sell..."}]
  ```
</ParamField>

<ParamField body="features" param-type="nested" showLocation="false">
  A list of features associated with this property. Each feature contains: `key` (label) and `value` (list of possible values).


  **Example:**
  ```json
  "features": [{"key": "Community District", "value": ["318"]}]
  ```
</ParamField>

<ParamField body="fees" param-type="nested" showLocation="false">
  A list of various fees that have been seen for this property. Each fee contains: `amountMax`, `amountMin`, `currency`, `dateSeen`, `sourceURLs`, `type`.

  **Example:**
  ```json
  "fees": [{"amountMax": 11, "amountMin": 11, "currency": "USD", "dateSeen": ["2016-08-07T18:07:05Z"], "sourceURLs": ["https://..."], "type": "Cleaning Fee"}]
  ```
</ParamField>

<ParamField body="floorSizeValue" param-type="text" showLocation="false">
  The size of the walkable area of the property. For apartments and condos, this is the size of the property. For homes and buildings, this is the combined size of each floor and does not include lot size.


  **Example:**
  ```json
  "floorSizeValue": 1000
  ```
</ParamField>

<ParamField body="floorSizeUnit" param-type="text" showLocation="false">
  The units of measurement for `floorSizeValue`.


  **Example:**
  ```json
  "floorSizeUnit": "Sq. ft"
  ```
</ParamField>

<ParamField body="hours" param-type="nested" showLocation="false">
  The hours of operation for property. Each hour object contains: `day` and `hour`.


  **Example:**
  ```json
  "hours": [{"day": "Saturday", "hour": "10:00am-5:00pm"}]
  ```
</ParamField>

<ParamField body="imageURLs" param-type="text" showLocation="false">
  A list of image URLs for this property.


  **Example:**
  ```json
  "imageURLs": ["http://image.rent.com/..."]
  ```
</ParamField>

<ParamField body="keys" param-type="text" showLocation="false">
  A list of internal Datafiniti identifiers for this property. The `keys` field is used to merge raw data from individual sources into the master Datafiniti record.


  **Example:**
  ```json
  "keys": ["us/ca/nhills/9411brightwoodct"]
  ```
</ParamField>

<ParamField body="languagesSpoken" param-type="text" showLocation="false">
  A list of the languages spoken at this property.


  **Example:**
  ```json
  "languagesSpoken": ["French", "English"]
  ```
</ParamField>

<ParamField body="latitude" param-type="text" showLocation="false">
  The latitude coordinates for this property.


  **Example:**
  ```json
  "latitude": "32.860858"
  ```
</ParamField>

<ParamField body="leasingTerms" param-type="nested" showLocation="false">
  A list of leasing terms that have been associated with this property. Each leasing term contains: `dateSeen`, `sourceURLs`, `value`.

  **Example:**
  ```json
  "leasingTerms": [{"dateSeen": ["2016-07-30T20:41:01Z"], "sourceURLs": ["https://..."], "value": "12 Months"}]
  ```
</ParamField>

<ParamField body="listingName" param-type="text" showLocation="false">
  The name of the property listing. This isn't the name of the building. It's a descriptive name supplied for the listing itself.


  **Example:**
  ```json
  "listingName": "Antico Appartamento"
  ```
</ParamField>

<ParamField body="longitude" param-type="text" showLocation="false">
  The longitude coordinates for this property.


  **Example:**
  ```json
  "longitude": "-96.735808"
  ```
</ParamField>

<ParamField body="lotSizeValue" param-type="text" showLocation="false">
  The lot size for the property. This is the ground area covered by the property.


  **Example:**
  ```json
  "lotSizeValue": 2219
  ```
</ParamField>

<ParamField body="lotSizeUnit" param-type="text" showLocation="false">
  The units of measurement for `lotSizeValue`.


  **Example:**
  ```json
  "lotSizeUnit": "Sq. ft"
  ```
</ParamField>

<ParamField body="managedBy" param-type="nested" showLocation="false">
  A list of companies or people that have managed this property. Each managedBy object contains: `dateSeen`, `sourceURLs`, `value`.

  **Example:**
  ```json
  "managedBy": [{"dateSeen": ["2016-03-03T15:02:13Z"], "sourceURLs": ["http://..."], "value": "ROM Investments"}]
  ```
</ParamField>

<ParamField body="mlsNumber" param-type="text" showLocation="false">
  The MLS number for the property. [Learn more about MLS numbers](https://en.wikipedia.org/wiki/Multiple_listing_service).


  **Example:**
  ```json
  "mlsNumber": "16177277"
  ```
</ParamField>

<ParamField body="nearbySchools" param-type="nested" showLocation="false">
  A list of schools near this property. Each school may contain: `assigned`, `address`, `city`, `country`, `dateSeen`, `distanceValue`, `distanceUnit`, `faxes`, `gradeLevels`, `name`, `phones`, `postalCode`, `province`, `sourceURLs`.

  **Example:**
  ```json
  "nearbySchools": [{"dateSeen": ["2017-06-11T11:40:54Z"], "distanceUnit": "mi", "distanceValue": 5.3, "name": "Holbrook High School", "sourceURLs": ["http://..."]}]
  ```
</ParamField>

<ParamField body="neighborhoods" param-type="text" showLocation="false">
  A list of neighborhoods where this property is located. Sometimes an area can belong to multiple neighborhoods, which is why this field is a list.


  **Example:**
  ```json
  "neighborhoods": ["Canarsie"]
  ```
</ParamField>

<ParamField body="numBathroom" param-type="text" showLocation="false">
  The number of bathrooms in this property.


  **Example:**
  ```json
  "numBathroom": 2
  ```
</ParamField>

<ParamField body="numBedroom" param-type="text" showLocation="false">
  The number of bedrooms in this property.


  **Example:**
  ```json
  "numBedroom": 3
  ```
</ParamField>

<ParamField body="numFloor" param-type="text" showLocation="false">
  The number of floors or stories in this property.


  **Example:**
  ```json
  "numFloor": 2
  ```
</ParamField>

<ParamField body="numPeople" param-type="text" showLocation="false">
  The number of people that can occupy this property.


  **Example:**
  ```json
  "numPeople": 6
  ```
</ParamField>

<ParamField body="numRoom" param-type="text" showLocation="false">
  The number of rooms in this property.


  **Example:**
  ```json
  "numRoom": 5
  ```
</ParamField>

<ParamField body="numUnit" param-type="text" showLocation="false">
  The number of units available in this property.


  **Example:**
  ```json
  "numUnit": 1
  ```
</ParamField>

<ParamField body="parking" param-type="text" showLocation="false">
  A list of details regarding parking at this property.


  **Example:**
  ```json
  "parking": ["Garage"]
  ```
</ParamField>

<ParamField body="paymentTypes" param-type="text" showLocation="false">
  A list of payment options accepted by the property.


  **Example:**
  ```json
  "paymentTypes": ["Visa"]
  ```
</ParamField>

<ParamField body="people" param-type="nested" showLocation="false">
  A list of known people associated with or employed by this property. Each person may contain: `dateSeen`, `email`, `name`, `phone`, `title`.


  **Example:**
  ```json
  "people": [{"name": "Pulcano Yvonne", "title": "Owner"}]
  ```
</ParamField>

<ParamField body="petPolicy" param-type="text" showLocation="false">
  The pet policy for the property.


  **Example:**
  ```json
  "petPolicy": "Pets are not allowed."
  ```
</ParamField>

<ParamField body="phones" param-type="text" showLocation="false">
  A list of phone numbers for the property.


  **Example:**
  ```json
  "phones": ["5058982700"]
  ```
</ParamField>

<ParamField body="postalCode" param-type="text" showLocation="false">
  The postal or zip code of the property.


  **Example:**
  ```json
  "postalCode": "75231"
  ```
</ParamField>

<ParamField body="prices" param-type="nested" showLocation="false">
  A list of prices for this property. Price objects may contain: `amountMin`, `amountMax`, `availability`, `comment`, `currency`, `date`, `dateAdded`, `dateSeen`, `dateValidStart`, `dateValidEnd`, `isSale`, `isSold`, `minStay`, `period`, `pricePerSquareFoot`, `sourceURLs`.

  **Example:**
  ```json
  "prices": [{"amountMax": 289000, "amountMin": 289000, "currency": "USD", "dateSeen": ["2017-10-09T00:00:00Z"], "isSale": "false", "isSold": "false", "pricePerSquareFoot": 197, "sourceURLs": ["http://..."]}]
  ```
</ParamField>

<ParamField body="propertyTaxes" param-type="nested" showLocation="false">
  A list of property taxes that have been seen for this property. Each property tax includes: `amount`, `currency`, `dateSeen`, `sourceURLs`.

  **Example:**
  ```json
  "propertyTaxes": [{"amount": 1582, "currency": "USD", "dateSeen": ["2016-08-09T09:34:15Z"], "sourceURLs": ["http://..."]}]
  ```
</ParamField>

<ParamField body="propertyType" param-type="text" showLocation="false">
  A classification for the property type.


  **Example:**
  ```json
  "propertyType": "Condo"
  ```
</ParamField>

<ParamField body="province" param-type="text" showLocation="false">
  The province or state for this property.


  **Example:**
  ```json
  "province": "TX"
  ```
</ParamField>

<ParamField body="reviews" param-type="nested" showLocation="false">
  A list of reviews for this property. Review objects may contain: `date`, `dateAdded`, `dateSeen`, `doRecommend`, `rating`, `sourceURLs`, `text`, `title`, `userCity`, `username`, `userProvince`.

  **Example:**
  ```json
  "reviews": [{"dateAdded": "2016-08-17T18:12:03Z", "dateSeen": ["2016-08-17T00:00:00Z"], "sourceURLs": ["https://..."], "text": "We love this place a lot!", "username": "Yuwei"}]
  ```
</ParamField>

<ParamField body="rules" param-type="text" showLocation="false">
  A list of rules for living or staying at this property.


  **Example:**
  ```json
  "rules": ["No smoking"]
  ```
</ParamField>

<ParamField body="sourceURLs" param-type="text" showLocation="false">
  A list of URLs used to generate data for this property.


  **Example:**
  ```json
  "sourceURLs": ["https://www.airbnb.com/rooms/13392408"]
  ```
</ParamField>

<ParamField body="statuses" param-type="nested" showLocation="false">
  A list of statuses for this property. Each status includes: `dateSeen`, `isUnderContract`, `sourceURLs`, `type`. See [Possible Values for Property Fields](/v3/docs/possible-values-for-property-fields) for status types.

  **Example:**
  ```json
  "statuses": [{"dateSeen": ["2017-10-10T20:12:16Z"], "isUnderContract": "false", "sourceURLs": ["http://..."], "type": "Sold"}]
  ```
</ParamField>

<ParamField body="taxID" param-type="text" showLocation="false">
  The tax ID for the property.


  **Example:**
  ```json
  "taxID": "12345678"
  ```
</ParamField>

<ParamField body="unavailableDates" param-type="nested" showLocation="false">
  A list of dates this property is unavailable (typically for rent). Each unavailability contains: `dateSeen`, `endDate`, `sourceURLs`, `startDate`.

  **Example:**
  ```json
  "unavailableDates": [{"dateSeen": ["2016-05-29T00:00:00Z"], "endDate": "2016-04-29T00:00:00Z", "sourceURLs": ["http://..."], "startDate": "2016-04-29T00:00:00Z"}]
  ```
</ParamField>

<ParamField body="websiteIDs" param-type="text" showLocation="false">
  A list of website IDs for this property. Website IDs are tied to a specific online listing source and are the unique identifier used by that website.


  **Example:**
  ```json
  "websiteIDs": ["domain.com-123", "website.com-4567"]
  ```
</ParamField>
